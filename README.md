# myplotlib

`matplotlib` binder with custom styles.

### installation

```shell
pip install git+https://github.com/haykh/myplotlib.git@master
```

### usage

```python
import myplotlib
# initialize styles:
myplotlib.load()

import myplotlib.plots as mypl
# type for docstring:
mypl? 
```

### requirements

* `matplotlib>=3.0.0`, `numpy`, `scipy`
* `latex` (install separately)
